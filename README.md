# Aerial Robotics PhD Preparation

This repository documents my preparation for graduate research in aerial robotics, autonomous systems, and intelligent control.

The goal is to build a solid theoretical and practical foundation before starting my master's research, with a long-term objective of pursuing a PhD in aerial robotics.

---

## Research Roadmap

### Phase 1 — Fundamentals
- Linear Systems
- State-Space Modeling
- PID Control
- Kalman Filtering

### Phase 2 — Advanced Control
- Model Predictive Control (MPC)
- Trajectory Tracking

### Phase 3 — Learning-Based Control

- Reinforcement Learning

### Phase 4 — Research Topics
- Quadrotor Dynamics
- Modular Aerial Robots
- Morphing UAVs
- Fault-Tolerant Flight

## Learning Workflow

For every topic:

1. Study the theoretical concepts.
2. Implement a simple example in Python or MATLAB.
3. Read related research papers.
4. Write a one-page summary.
5. Identify limitations and open research questions.

---

## Paper Summary Template

Each paper summary includes:

- Problem
- Proposed Method
- Assumptions
- Main Contributions
- Limitations
- Possible Future Work
- Personal Notes

---

## Weekly Progress

Every week:

- Study report
- Paper summaries
- Code implementation
- Questions and observations

Every month:

- 10-minute presentation
- Code demonstration
- Literature review
- 2–3 possible research questions
